### 日常操作

静态 ip 配置

```
# cd /etc/sysconfig/network-scripts/
# 找到对应的网卡
# vim  ifcfg-eth0
NAME="eth0"
ONBOOT="yes"
NETBOOT="yes"
BOOTPROTO="none"
TYPE="Ethernet"
IPADDR=10.2.0.20
GATEWAY=10.2.0.1
PREFIX=24
DNS1=202.96.209.133
DNS2=114.114.114.114
DEFROUTE="yes"

# 重启网络服务
# service network restart

```



dig

```
dig sparkpay.top MX    # 查询邮件服务器
dig sparkpay.top NS    # 查询域名授权服务器
dig sparkpay.top TXT   # 查询文本记录（常用于验证域名所有权、SPF/DKIM 记录）
dig sparkpay.top ANY   # 尝试查询所有记录（由于安全限制，部分服务器可能不返回）

#  排查解析路径
dig +trace sparkpay.ai

# 指定服务器
dig @8.8.8.8 sparkpay.ai

# 反向解析
dig -x 91.195.240.94
```

nslookup

```
# 指定类型
nslookup --type=txt sparkpay.top
```
