# 电力基础

一、 电力如何进入你家？

从电力局（电网）到你家门口，电力通常经历以下几个关键节点：

1. **高压输电（远距离传输）：** 发电厂产生的电通过**特高压（1000kV/±800kV）**或超高压线路跨省传输。
2. **区域变电站（初步降压）：** 进入城市边缘后，电压被降至 **110kV** 或 **35kV**。
3. **小区配电房/变压器（最后冲刺）：** 这是关键的一步。电力局的 **10kV** 高压线进入小区的配电房，通过变压器降压为 **380V（三相电）**。
4. **入户线（最后100米）：**
   - **普通住宅：** 从变压器引出三根火线和一根零线（三相四线）。电工会将这三根火线**平均分配**给不同的住户。比如 A 户拿 L1 火线，B 户拿 L2，C 户拿 L3。这就是为什么你家是 220V 的原因——你只拿了其中一相。
   - **大户型/别墅：** 有些家庭由于电器极多（中央空调、地暖、大功率烤箱），会申请**三相入户**，直接拉进四根或五根线。



 “三相电” vs “单相电“

| **特性**     | **单相电 (Single-phase)**          | **三相电 (Three-phase)**         |
| ------------ | ---------------------------------- | -------------------------------- |
| **导线数量** | 2根线（1火1零）或3根（含地线）     | 4根线（3火1零）或5根（含地线）   |
| **标准电压** | **220V**                           | **380V**                         |
| **适用场景** | 普通照明、电视、小空调、手机充电   | 工业机床、电梯、大功率中央空调   |
| **动力感**   | 像一个人推车，推力有间歇（正弦波） | 像三个人轮流推车，动力平稳且强大 |

###  为什么叫“三相电”？

它是三根火线，每一根的电流波形互相错开 120°。如果测量两根火线之间的电压，就是 **380V**；如果测量任意一根火线与零线之间的电压，就是你熟悉的 **220V**。

电线的选择主要取决于**电流（A）**，而电流由**功率（W）**决定。因为三相电（380V）和单相电（220V）的电压不同，同样的功率，三相电所需的电线往往更细。
